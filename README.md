# Brick Smasher

A fresh take on the classic Breakout formula, featuring Super Smash Bros.-inspired knockback physics.

## About

Brick Smasher reimagines the brick-breaking genre by replacing instant destruction with satisfying physics-based knockback. Instead of breaking on contact, bricks accumulate damage and get launched further with each hit. Your goal: knock them completely off the screen to eliminate them.

## Features

- **Knockback Physics** - Bricks react to impacts with realistic physics. The harder you hit, the further they fly.
- **Damage System** - Bricks build up damage over time, making them increasingly vulnerable to being launched.
- **Variety of Equipment** - Unlock and choose from multiple balls and paddles, each with unique properties.
- **Skill-Based Gameplay** - Master precise timing and positioning to maximize your knockback power.

## Platforms

- Mobile (iOS & Android) - Primary target
- Additional platforms planned for future releases

## Development

Built with Unity 2D using the Universal Render Pipeline.

### Requirements

- Unity 2022.3 LTS or newer
- Universal RP package
- Input System package

### Getting Started

1. Clone the repository
2. Open the project in Unity Hub
3. Open `Assets/Scenes/SampleScene.unity`
4. Press Play to test
